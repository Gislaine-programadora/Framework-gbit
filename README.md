## 💡 O que é o Gbit Framework?

**Gbit** é uma poderosa ferramenta **open-source** projetada para desenvolvedores modernos. Com um único comando, você cria uma estrutura completa de projeto — incluindo **backend**, **frontend** e **smart contracts** — pronta para construir e fazer deploy.

```bash
npx create-gbit-app "meu-projeto"
```

## 🧱 Gbit Build - Ferramenta de Compilação

<div align="center">
  <img src="assets/gbit-npm.png" alt="Gbit Build Logo" width="150" height="150" />
</div>

### Instale e use o **gbit-build** para compilar seus projetos TypeScript/Vite:

```bash
# Instalar
npm install --save-dev gbit-build

# Build completo
npx gbit-build

# Build apenas frontend
npx gbit-build --only-frontend

# Build sem Docker
npx gbit-build --skip-docker

# Modo verbose
npx gbit-build --verbose

# Dry run (simular build)
npx gbit-build --dry-run
```

### Comandos Disponíveis

| Comando | Descrição |
|---------|-----------|
| `npx gbit-build` | Build completo (frontend + backend + Docker) |
| `--only-frontend` | Compila apenas o frontend |
| `--skip-docker` | Ignora geração de imagem Docker |
| `--verbose` | Mostra logs detalhados |
| `--dry-run` | Simula o build sem executar |

O **gbit-build** automatiza:
- ✅ Build do frontend (Vite/React/Next.js)
- ✅ Compilação do backend
- ✅ Geração de imagem Docker
- ✅ Otimização de assets
- ✅ Minificação de código

## 🚦 Começando

### 1. Criar Projeto

```bash
npx create-gbit-app meu-app
```

### 2. Escolher Stack

Durante a instalação, você escolhe:
- ✅ Frontend (React/Next.js/Vite)
- ✅ Backend (Node/Python)
- ✅ Banco de Dados (PostgreSQL/MongoDB/MySQL)
- ✅ Smart Contracts (Solidity/Vyper) - Opcional

### 3. Iniciar Desenvolvimento

```bash
cd meu-app

# Instalar dependências
npm install

# Rodar desenvolvimento
npm run dev
```

### 4. Build & Deploy

```bash
# Build de produção
npx gbit-build

# Deploy (Vercel/Docker)
npm run deploy
```
